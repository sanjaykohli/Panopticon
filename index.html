Technical Overview

Panopticon demonstrates client-side data collection techniques. It illustrates how modern web applications can identify and track users without utilizing cookies, relying instead on browser APIs, hardware differences, and network behaviors.

Canvas Fingerprinting
Critical Risk
Technique

The browser renders hidden text and gradients. Subtle differences in GPU rendering, anti-aliasing, and OS font handling produce unique pixel data.

Privacy Impact

Creates a persistent identifier (hash) that remains consistent even after clearing cookies or utilizing private browsing modes.

WebRTC IP Leak
Critical Risk
Technique

WebRTC establishes peer-to-peer connections. To facilitate this, it may expose local LAN IP addresses and, in some configurations, the true public IP behind a VPN.

Privacy Impact

Reveals internal network architecture and potentially bypasses VPN protections to disclose physical location.

Hardware & Audio Context
Medium Risk
Technique

Scripts analyze CPU core count, RAM estimates, battery behavior, and audio signal processing characteristics.

Privacy Impact

Contributes to a composite fingerprint. While individual metrics are not unique, the aggregate profile is highly specific.
